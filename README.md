## Requirements
+ `Python` >= 3.9.7
+ `Pip` >= 21.2.3
  

## Usage
Clone our source code
```sh
$ git clone https://github.com/nh0znoisung/Sokoban
$ cd Sokoban
```

Install Dependencies

```sh
$ pip install -r requirements.txt
```

Run program

```sh
$ python main.py
```
